Data, variables, values, class, logic, AND, OR, NOT, data frame, data set, feature, observation, histogram, scale, rescale, table, reshape, pivot, query

Variables

The basic unit of data is the variable. We create variables and assign them values.

var.number <- 1 #integers are descrete, decimals are continuous
var.logical <- TRUE
var.date <- as.Date("12/25/2018", "%m/%d/%Y")
var.vector <- c(1,2,3,4)
var.character <- "hello"
var.factor <- as.factor(var.vector)

After we create variables, we can access the variables by name and review their values

var.number
[1] 1
var.logical
[1] TRUE
var.date
[1] "2018-12-25"
var.vector
[1] 1 2 3 4
var.character
[1] "hello"
var.factor
[1] 1 2 3 4
Levels: 1 2 3 4

Each variable has a class. The variable class is determined by the data assigned to it at creation.

#class type
class(var.number)
[1] "numeric"
class(var.logical)
[1] "logical"
class(var.date)
[1] "Date"
class(var.vector)
[1] "numeric"
class(var.character)
[1] "character"
class(var.factor)
[1] "factor"

Logic

Source: historyinformationliteracy.wordpress.com

socrates.mortal <- TRUE
socrates.immortal <- FALSE
socrates.mortalANDimmortal <- socrates.mortal && socrates.immortal
socrates.mortalANDimmortal
[1] FALSE
socrates.mortalORimmortal <- socrates.mortal || socrates.immortal
socrates.mortalORimmortal
[1] TRUE
if (socrates.mortalANDimmortal){
  print("That's impossible!")

  print("No one is both mortal and immortal")
}

Data Frames

A data frame is fundamental entity in data science. It’s the same as data you are used to seeing in a comma separated file (CSV) or an Excel spreadsheet. The primary representation of a dataset is a data frame.

df <- as.data.frame(airquality)
df[1:5,]
  Ozone Solar.R Wind Temp Month Day
1    41     190  7.4   67     5   1
2    36     118  8.0   72     5   2
3    12     149 12.6   74     5   3
4    18     313 11.5   62     5   4
5    NA      NA 14.3   56     5   5

A feature is any column in the data frame.

df$Temp[1:5]
[1] 67 72 74 62 56

An observation is any row from the data frame. For example, the 4th observation corresponds to the vector in the 4th row of the data frame.

n=3
df[n,]
  Ozone Solar.R Wind Temp Month Day
3    12     149 12.6   74     5   3

A distribution describes how often particular values of a feature occur. The following histogram shows the distribution of the temperature feature. The values of the temperature feature range from 50 degrees to 100 degrees. Most often, we see temperatures between 75 degrees and 85 degrees.

hist(df$Temp)

The temperature feature is on a scale of 50 degrees to 100 degrees, but we can rescale the temperature feature so that the values range from, for example, 0 to 1.

#function to check if a package is installed
is_installed <- function(mypkg) is.element(mypkg, installed.packages()[,1])
#install and include the scales package
if(!is_installed("scales")){install.packages("scales")}
library(scales)
#rescale the temperature values
zero.to.one <- c(0,1)
temperatures0to1 <- rescale(df$Temp, zero.to.one)
#use a histogram to see the new scale of the temperature values
hist(temperatures0to1)

We can reshape a data set so that the observations and features are different. Take, for example a data set that shows observations of Titanic survivors.

df.titanic <- as.data.frame(Titanic)
df.titanic[1:5,]
  Class    Sex   Age Survived Freq
1   1st   Male Child       No    0
2   2nd   Male Child       No    0
3   3rd   Male Child       No   35
4  Crew   Male Child       No    0
5   1st Female Child       No    0

We can reshape (or pivot) the Titanic data set to show the number of survivors by sex and age. The result is a table with columns that correspond to the values of the Age feature, and rows that correspond to the values of the Sex feature.

#install and include the reshape package
if(!is_installed("reshape")){install.packages("reshape")}
library(reshape)
df.titanic.pivot <- cast(df.titanic, Sex~Age, sum, value = "Freq")
df.titanic.pivot
     Sex Child Adult
1   Male    64  1667
2 Female    45   425
